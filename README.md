# SMART TASK MANGER

A Flask app that uses smart prioritization to guide users on important taks to complete on a daily basis, improving productivity.

## Features
+ add/edit/delete taks
+ daily summary dashboard
+ Priority algorithm
+ SQLite Database

## Setup
1. Clone the repo
2. Create a virtual environment
3. Install dependencies: `pip install -rrequirements.txt`
4. Run the app: `python app.py`
